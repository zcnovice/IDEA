下面关于数组的说法中，错误的是?(2分)

A.数组的长度是不可变的

B.数组可以在内存空间连续存储任意一组数据

C.数组必须先声明，然后才能使用

D.数组元素的类型一致

选**B**



1. **选项 C 的正确性**
   Java 要求数组变量必须**先声明**（如 `int[] arr;`），才能被使用（如赋值或访问元素）。
   即使你写成 `int[] arr = {1,2,3,4,5};`，本质上也是：  
   - **声明数组变量**：`int[] arr`
   - **初始化数组内容**：`{1,2,3,4,5}`
     这属于“声明后立即初始化”，仍然符合“先声明后使用”的规则。
